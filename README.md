<h3>功能列表</h3>

```
1. 数据收集
    1.1. 性能信息收集
    1.2. 错误信息收集

2. 数据汇总
    2.1. 收集到数据库
    2.2. 选择性收集（前台控制/后台控制）
    2.3. 分类收集（按来源网站，白名单/黑名单）
    
3. 信息研判
    3.1. 信息统计（访问量、性能变化量、错误量等）
    3.2. 预置联系信息、处理办法
    
4. 警报发出
    4.1. 当访问量高于/低于一定预期后，自动发出警报
    4.2. 当错误量突然升高后，自动发出警报
    4.3. 警报发出：短信/右键
```
